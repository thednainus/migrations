# Estimation of migration rates

In this repository you can find the scripts created to estimate migration rates using different methods.

* **Analysis_SD** directory contains the files adapted to the San Diego data. For a complete descrition of scripts, see wiki.
* **Fit** directory contains the the best runs for each fit I did using different combination of parameter values using this [script](https://github.com/thednainus/SD_Benefits/blob/main//Analysis_SD/fit/Sampling_LHS.R)
* **CI_migrationRates_plots** directory contains the plots published in the paper (see citation below).


## Citation

Nascimento FF, Mehta SR, Little SJ, Volz EM. (2025) [Robust phylodynamic inference and model specification for HIV transmission dynamics](https://www.sciencedirect.com/science/article/pii/S1755436525000349?via%3Dihub). Epidemics, 52: 100846.
